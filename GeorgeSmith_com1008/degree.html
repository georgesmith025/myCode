Sheffield Computer Science degree course

These are the modules I have been studying during the autumn semester of my first year. You can check out more information on further study in the computer science department here.

Introduction to Software Engineering

This module introduces software engineering concepts and techniques that are needed to develop systems to meet requirements and overcome problems. In addition, we are coding with the language Ruby designed to bridge the gap between code and psuodocode.

Foundations of Computer Science

This module develops mathematical concepts and techniques that are of foundational importance to computing. This includes topics such as set theory and boolean algebra

Java programming

This module introduces computer programming through the Java programming language. This covers programming concepts like object orientation and teaches the use of the Sheffield package for certain techniques.

Machines and Intelligence

This module introduces the history, ideas, struggles, and modern day progress with Artificial Intelligence. In addition, we practise AI techniques with the adorable, little MIRO robots.

Devices and Networks

This module introduces computers in their most basic form made up of logic gates. This means other topics revolve around binary, simplifying logic circuits, and bridging the gap between theory and electrical implementation.

Web and Internet Technology

This module introduces the basics of networks and internetworking, information security, and the technologies that are used in Web development, with an emphasis on building a simple interactive website using HTML, CSS and JavaScript.
